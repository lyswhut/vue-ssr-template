# vue-ssr-template

A vue-ssr based scaffolding.

Scaffolding mainly integrates the following packages:

* vue 2.x
* vue-router
* vuex
* axios
* webpack 4.x
* babel 7.x
* autoprefixer
* eslint
* less
* stylus
* pug

## How to use

```bash
# dev mode
npm run dev

# production build
npm run build

# eslint check
npm run lint

# eslint fix
npm run lint-fix

```
